# NatureSoundsWithMeditaions

NatureSoundsWithMeditaionsは時間がない人へ短時間で、どこでも瞑想ができることをコンセプトにしたアプリケーションです。

▼サービス URL

https://nature-sounds01.com/

## 概要


## 主な機能
* 動画投稿機能
* 瞑想記録機能
* 動画タグ検索機能

## ユーザー機能
* ログイン機能（ゲストログイン可能）
* 瞑想記録管理機能

## 使用技術

### フロントエンド
* HTML
* CSS
* JavaScript
* DaisyUI

### バックエンド
* Rails(7.0.6)
* NginX
* RSpec

### データベース
* MySQL

* docker-compose

### 外部サービス
